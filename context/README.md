# Context Directory

This directory is reserved for future machine-readable context exports, derived summaries, and auxiliary views that mirror root-level manifests without replacing them.

Phase 1 keeps the canonical context files at the repository root:

- `AGENTS.md`
- `active_focus.md`
- `lab_manifest.yaml`
- `rigor_policy.md`
- `source_index.yaml`
- `notation_registry.yaml`
- `gap_log.md`

Later phases may add derived files here if they can be regenerated safely.
